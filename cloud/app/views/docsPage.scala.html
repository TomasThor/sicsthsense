SicsthSense is based on the concept of resources, representing some type of device or source of informtion. Data is then pulled or pushed from these resources to create streams of data. This data is stored by SicsthSense and made available for further processing by you and others. Resources provide a definition of where and how a source of data is accessed, and then streams of data points are populated from this resource.

The first step is to create an account. You can then add your own data resources and start collecting their data. You can even track other people's resources.

If you are having any problems, please feel free to contact us. We are working on improving SicsthSense and your feedback/requests can only help this.


Web Interface

The easiest and quickest way to interact with SicsthSense is through the web interface. You can add resources, define how to parse your data and view graphs of the data that collects in your data streams.


Programatic API

You can also access all of SicsthSense through a simple RESTful API (i.e., using GET, POST on URLs). This manner of access is best for programmers and other computing devices that may be wanted to access your stored data.
